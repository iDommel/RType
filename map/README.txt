Map are composed of up to 600 lines of 17 caracters.
The last line define the end of the Map.
YOU NEED TO HAVE AN EMPTY LINE AT THE END OF THE FILE !

List off caracters:

P:  Player spawn

*:  void
a:  borderBlue Ground

1:  Basic Enemy
2:  Second Enemy
3:  Bonus Enemy
4:  Strong Enemy
5:  Turret

f-j: Same for Brown mobs

k-o: Same for Green

A-B:  Boss Red Brown Green
